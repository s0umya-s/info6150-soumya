Mobile First Design

A responsive/adaptive site

  • Design the mobile version first

Why Mobile First Design

  • Easier to expand than collapse
  • Encourages clear designs
  • Defines priorities

Mobile First CSS

Not just the design, the implementation

  • Write HTML, thinking mobile
  • Write CSS for mobile design
  • Media queries apply larger settings
    • tablet/desktop/etc
  • Mobile usually more simple CSS
    • Mostly stacked vertical blocks
  • Avoids "undoing" CSS

When Mobile First CSS isn't more simple

A "hamburger menu"

  • Or other "collapsing" navigation structure

Reverse of previous example

  • Collapsing sections/menus are more complex CSS
  • If written Mobile First CSS
    • Desktop "undoes" the complex CSS

You have to consider and decide at start

  • What is the most simple set of CSS options?
